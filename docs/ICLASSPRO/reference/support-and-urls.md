## Office Portal Settings Navigation

Quick reference for finding settings in the staff-facing Office Portal:

| What You're Looking For | Navigation Path |
|------------------------|-----------------|
| General Settings | SETTINGS > SETUP > GENERAL SETTINGS |
| Class Settings | SETTINGS > SETUP > CLASS SETTINGS |
| Customer Portal Settings | SETTINGS > CUSTOMER PORTAL |
| Class Registration Options | SETTINGS > OTHER SETTINGS > CLASS REGISTRATION |
| Marketing and Branding | SETTINGS > OTHER SETTINGS > MARKETING AND BRANDING |
| Party Configuration | SETTINGS > PARTIES |
| Skill Tracking Options | SETTINGS > SETUP > GENERAL SETTINGS > SKILL TRACKING |
| Autopilot Workflows | AUTOPILOT > WORKFLOWS |
| Add a Workflow | AUTOPILOT > WORKFLOWS > ADD A WORKFLOW |
| Location Management | SETTINGS > SETUP > LOCATIONS |
| Enable Party Booking | SETTINGS > CUSTOMER PORTAL > OTHER SETTINGS > GENERAL SETTINGS |
| Communication Templates | (varies by template type — check Communications section in KB) |
